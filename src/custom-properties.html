Custom Properties

Chassis includes many CSS Custom Properties to give developers the ability to use computed values in their CSS. Developers can also add their own through a Chassis Theme File.

Layout

Property Returns
--layout-min-width layout.minWidth as specified in your Build Configuration
--layout-max-width layout.maxWidth as specified in your Build Configuration
--layout-min-width layout.gutter as specified in your Build Configuration
--layout-min-width Width Constraint minimum gutter value
--layout-max-width Width Constraint maximum gutter value
