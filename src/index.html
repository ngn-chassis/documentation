Chassis Components

Chassis includes a suite of themeable components which make UI development fast and painless. These components are customizable and extendable using the @chassis new or @chassis extend mixins.

Chassis components are designed to work seamlessly with Chassis' automated typography-based layout system. The size of each component can be changed using the @chassis font-size mixin, or it can be sized manually if desired.

Form Controls

The best way to use Chassis Form Controls is with author-control. This is a wrapper component which provides a simple API for interacting with all types of form controls. It adds a number of workflow-enhancing features. Chassis Form Controls can still be used without author-control if desired.
